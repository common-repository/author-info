Author Info is a widget which shows the author's bio on the widget

== Description ==

This plugin will show the author's bio together with author's photograph on the blog footer or sidebar. 
 
== Frequently Asked Questions == 

= How to set up the css style? = 
 
Click on the 'Author Info' tab under Settings to do the css setting inside the 'Author Settings Page'.   

= Where to write our bio? = 

Grab the Author Info widget under the Widgets page to any of the Sidebar and fill in the respective fields.

= Can I change the information later on? =

Sure, anytime, just re-enter your new information and click on the save button of the widget.

== Installation ==

1) Upload Author Info plugin to your blog, then activate it.
2) Then follow the instructions under FAQ to use this widget.
